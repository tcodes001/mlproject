Welcome to the Student Performance Predictor

This project was born out of a simple question: “What if we could identify when a student might be struggling in math — before it’s too late?”

As someone deeply interested in both data science and education, I built this tool to explore how machine learning can be used not just for predictions, but for early intervention.

The model uses details like a student's reading and writing scores, background, and learning environment to predict their math score. But more importantly, it also tells you why the model made that prediction — through a personalized, human-readable explanation.

Whether you’re an educator, a parent, or just someone curious about AI, this tool shows how data can help us make smarter, more empathetic decisions in the classroom.
